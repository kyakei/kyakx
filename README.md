## **kyakx**

## **Installation**
### **Using Pipx (Recommended)**
```sh
pipx install kyakx
```
or

```sh
pip install kyakx
```

### **Manual Installation**
If you prefer running it manually:  
```sh
git clone https://github.com/kyakei/kyakx.git
cd kyakx
pip install -r requirements.txt
python3 cli.py
```
---
## **Usage**
Run the tool using:  
```sh
kyakx
```
Or, if running from source:  
```sh
python3 cli.py
```
